1. サービス概要

01 / 02は、キャスト・セラピストと店舗が登録できる業界専用SNSです。

  • 01 … 風俗専用SNS（キャスト向け：h-sns.net / 店舗向け：m-sns.net のLPから誘導）
  • 02 … メンズエステ専用SNS（セラピスト・店舗向け：m-sns.net）

「女の子が安心して発信できる場」を第一に設計。X（旧Twitter）のような凍結リスクや身バレの不安を抑えつつ、キャスト・セラピストの発信力が店舗の集客力につながる両立型SNSです。店舗も投稿機能で告知可能です。

こんなお悩みに（課題）

  • キャストにSNSをやってもらいたいが、Xの凍結が怖くて勧めづらい
  • 身バレが怖くて女の子が発信に消極的
  • 写メ日記だけでは新規の動きが弱い
  • 店のSNSだけでは限界で、結局キャストの発信に頼っている
  • イベント告知をもっと多くのユーザーに届けたい

→ 01/02なら「キャストの発信＋店舗の告知」で相乗効果を狙える、と訴求。

01/02が選ばれる4つのポイント（共通）

  • 凍結しない業界専用SNS … 積み上げた投稿・フォロワーが消えない
  • 身バレ対策 … 業界専門なので一般SNSのハードルを下げる
  • 投稿したくなるUI … 写真・動画・24時間で消えるストーリーズなど
  • 店舗も参加可能 … 店舗投稿で告知可能、キャストと合わせて露出最大化

店舗にとっての具体的メリット

  • ① 発信量アップ＝売上の土台UP … 安全な場があると発信が増え、指名・本指名・予約に直結
  • ② 広告費ゼロで露出増 … キャスト・店舗投稿がエリアの新着・人気・ストーリーズに表示
  • ③ 在籍安定・求人にもプラス … 「ファンを作りやすい店」はキャストから選ばれやすく、面接時のアピールにも

ご利用の流れ（店舗）

  1. 店舗アカウントを申請（店舗名・メール・公式サイトURLなど）→ 審査後ログイン情報送付
  2. 店舗情報を登録（ロゴ・説明・エリアなど）→ 店舗プロフィールページ作成
  3. 投稿開始（イベント・新人紹介など）
  4. 在籍女性にキャスト向けLPを共有 → 店舗ページに自動連携

4. キャスト・セラピスト向けLP（h-sns.net/lp/cast/）

※01（風俗）のキャスト向けを中心に記載。02（メンズエステ）は m-sns.net/lp/cast/ で同様の訴求。

キャッチコピー

「Xより安全で、もう凍結に怯えない。」
「もっと自分のファンを増やしたい」そんな女の子のための、キャスト主体のSNS。

訴求バッジ

  • 凍結リスクなし
  • 身バレリスク回避
  • 店移動もOK

課題（モヤモヤ）

  • SNSでの不安 … 写真の線引き・凍結・身バレの心配
  • ファン作りの悩み … 自分のペースで発信したいが方法がわからない
  • DM対応 … 営業以外のDM・リプ・アンチ対応など本業以外のストレス

01なら解決（4つ）

ポイント内容
「あなた」のファンが増える エリア別タイムライン・ランキングで新規に見つけてもらいやすい。指名UP・フォロワーは資産・凍結リスクなく訴求可能。
凍結の心配ゼロ 業界ルールに沿った投稿なら凍結なし。リットリンク代替のプロフページも。積み上げが無駄にならない。
風俗ユーザーだけに届く 見ているのは風俗に興味があるお客様だけ。効率よく集客・無駄なインプレッションゼロ。
めんどうなことナシ コメント機能なし→アンチ対応不要。DMは任意設定。投稿だけに集中できる。
